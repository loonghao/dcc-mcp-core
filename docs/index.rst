简介
----

DCC-MCP-Core 是一个为数字内容创建(DCC)应用程序设计的插件管理系统。它提供了一个灵活的框架，用于发现、加载和管理插件，并将插件信息以 AI 友好的结构化格式返回给 MCP 服务器。

主要功能
--------

* 插件发现和加载
* 插件函数调用
* 跨 DCC 应用程序的抽象层
* 为 AI 提供结构化的插件信息
* 使用 RPyC 实现远程过程调用
